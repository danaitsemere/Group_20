Input: A hashmap with letter grades to numeric values,list of student records each containing Student ID,
 INstructor Name, and Courses; each course includes Name,Instructor, Credits and Grade.

Process:

1. If student ID  is not in the student list record, indicate student is not found.
2. If a student ID is found, proceed with the following as needed.
3. If a course exists in the courses list Update the grade and confirm the update.
4. If the course does not exist, allow student to add a new Course, Instructor Name, Credit , and Grade.
5. Sum the weighted grades and Credits for courses with grades.
6. If no grades exist, indicate no grades are available.
7. Otherwise no grade exists, indicate no grades are available.

Output: Student report, Course Schedule, Course lists with detail,GPA values and Progress reports
